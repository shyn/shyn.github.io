Google语音识别API

Mar 27, 2015 · 1 minute read · Comments

https://github.com/gillesdemey/google-speech-v2

curl -X POST \
--socks5-hostname 127.0.0.1:1080 \
--data-binary @test.flac \
--user-agent 'Mozilla/5.0 (Macintosh; Intel Mac OS X 10_10_2) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/41.0.2272.104 Safari/537.36' \
--header 'Content-Type: audio/x-flac; rate=44100;' \
'https://www.google.com/speech-api/v2/recognize?output=json&lang=zh-CN&key=YourKey'

如何申请key？请参照原链接。 google服务器应该有单文件大小限制，不知道为多少，反正我发送的「内核恐慌」的一期录音文件（45.8MB），返回了失败的结果， 本来想利用这个做成文本，提取干货的，免得要听一个多小时。
